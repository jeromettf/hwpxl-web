hwpxl

한컴 HWP / HWPX / PDF 문서를 브라우저 안에서 미리 보고 Excel(.xlsx)로 내려받습니다.

100% 클라이언트 사이드 · 파일이 서버에 올라가지 않습니다
Python 런타임(Pyodide) 로드 중… 첫 로딩은 5~10초 걸릴 수 있습니다.
파일을 드래그하거나 클릭해 선택 (여러 개 가능) .hwp · .hwpx · .odt · .pdf · 각 50MB 이하

미리보기
어떻게 동작하나요?
  • 브라우저에서 Pyodide(웹어셈블리 Python)를 한 번 다운로드.
  • 같은 폴더의 hwpxl-*.whl 과 olefile, openpyxl, pdfminer.six 를 자동 설치.
  • 여러분이 올린 파일은 브라우저 메모리 안에서만 처리됩니다.
  • 여러 파일을 한 번에 올리면 결과 .xlsx 들이 자동으로 zip으로 묶여 다운로드됩니다.
지원 형식과 한계
  • HWP 5.x, HWPX, ODT, PDF (텍스트 기반)
  • 본문 문단·표·셀 병합 추출. PDF는 좌표 기반 표 추정.
  • 이미지·도형·수식은 포함되지 않습니다.
  • 암호화된 문서, 스캔(이미지) PDF는 처리할 수 없습니다.
